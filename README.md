# IMC_Calculator
Site hosted on Github: https://joaojoselimacerqueira.github.io/IMC_Calculator/

English version.
General: Study project, training how to do a IMC calculator, with interactions.
Technologies: CSS, HTML, JS.

My experience: 
1. I wanted a clean and objective interface to user, where they could just open the app and see straight the spots to calculate their IMC. Because people who look for a site who makes this math, usually already knows what IMC is, so let´s deliver the information they are really looking for. So, after results, the user would see a table with all IMC levels, as a way of get known about where they are and how to change that to theirs objectives. 
2. First, I built HTML initial structure, thinking of everything I would use, link with JS and CSS, and show on my website.
3. So I made JS code, all his logic is after the "calcular" button being clicked, all based in declarations, conditions, one main formula math and a table who communicate with CSS;
4. To finish, I did CSS, stylizing the main structure; then what happens after "calcular" button has been clicked, making the answer and table appearing with transitions; and making responsive to phone website's view.

Learn:
1. It was my first time making connections into tables in these technologies, the logic was not difficult, but I get a little stuck on how to transform logic on programming language to make it happen, hopefully, less than one hour I learned.
2. I got better at DOM manipulation, by practing.
3. I could learn more about how to make a responsive code.

Versão PT-BR.
Geral: Projeto de estudo, treinando como desenvolver uma calculadora de IMC com interações.
Tecnologias: CSS, HTML, JS.

Minha Experiência:
1. Eu planejei uma interface limpa e objetiva, onde o usuário pudesse abrir o app e visualizar diretamente os campos para calcular o IMC. Como quem busca esse tipo de site geralmente já sabe o que é o IMC, o foco foi entregar a informação desejada rapidamente. Após o resultado, o usuário visualiza uma tabela com todos os níveis de IMC para entender sua posição atual e como atingir seus objetivos.

2. Primeiro, construí a estrutura inicial em HTML, planejando tudo o que seria utilizado, a integração com JS e CSS, e o que seria exibido no site.
Em seguida, desenvolvi o código JS. Toda a lógica é disparada após o clique no botão "calcular", baseada em declarações, condicionais, uma fórmula matemática principal e uma tabela que se comunica com o CSS.

3. Para finalizar, fiz o CSS, estilizando a estrutura principal, as transições para a exibição da resposta e da tabela após o cálculo, além de tornar o site responsivo para dispositivos móveis.

Aprendizados:
1. Foi minha primeira vez criando conexões com tabelas nestas tecnologias. A lógica não foi difícil, mas precisei de um tempo para entender como transformá-la em código; felizmente, aprendi em menos de uma hora.

2. Melhorei minhas habilidades em manipulação de DOM através da prática.

3. Pude aprofundar meus conhecimentos em como criar um código responsivo.
